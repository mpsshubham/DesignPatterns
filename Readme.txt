With Design Patterns instead of code reuse, we get experience reuse
How to learn ?
Load your brain with design patterns and then recognize places in your design and existing applications.
Design pattern give us a shared vocabulary with other developers, easy communication with other developers
Elevates thinking at pattern level, instead gritty object level
